# PD2-Weapon-Stat-Randomizer
The base repository for the Payday 2 Weapon Stat Randomizer mod 

## About

This mod was created out of passion and tomfoolery of 2 close friends who have played this game so much that they need to mess up the balancing in order to have any fun anymore.

Its not meant to be a public release and we treat it as such. There is no support of updates and if something doesnt work as youd like then i hope you know Java. However any improvements and suggestions are welcome.

## Getting Started

### Prerequisites

- Payday 2 installed
- Java runtime environment installed

### Installation & Usage

1. Download the latest release of the PD2 Weapon Stat Randomizer from the [releases](https://github.com/nunokingler/PD2-Weapon-Stat-Randomizer/releases) section.
2. Check the instructions there
